  <implementation>
    ## Step 1: Add computePodSizes() function

    Export a new pure function `computePodSizes(playerCount: number, allowPodsOf3: boolean): { podSizes: number[]; byeCount: number }`.

    Implementation logic:
    - When `allowPodsOf3 === false`: `podSizes = Array(floor(n/4)).fill(4)`, `byeCount = n % 4`
    - When `allowPodsOf3 === true`:
      - n=3: `{ podSizes: [3], byeCount: 0 }`
      - n=5 (only unsolvable): `{ podSizes: [4], byeCount: 1 }`
      - n >= 4, n != 5: use remainder formula:
        - remainder 0: all 4s
        - remainder 1: `floor(n/4) - 2` fours + 3 threes
        - remainder 2: `floor(n/4) - 1` fours + 2 threes
        - remainder 3: `floor(n/4)` fours + 1 three
    - Place this function BEFORE generatePods in the file, after the existing helpers.

    ## Step 2: Modify greedyAssign() signature

    Change `greedyAssign(pool, numPods, history)` to `greedyAssign(pool, podSizes, history)` where `podSizes: number[]`.

    - Change `Array.from({ length: numPods }, ...)` to `Array.from({ length: podSizes.length }, ...)`
    - Change inner loop from `for (let slot = 1; slot < 4; slot++)` to `for (let slot = 1; slot < podSizes[podIdx]; slot++)`
    - Update pod iteration from `podIdx < numPods` to `podIdx < podSizes.length`

    ## Step 3: Modify generatePods() signature and body

    Add `allowPodsOf3: boolean = false` as third parameter (backward-compatible default).

    Key changes:
    - Validation: `const minPlayers = allowPodsOf3 ? 3 : 4`, check `activePlayers.length < minPlayers`
    - Error message: `Fewer than ${minPlayers} active players`
    - Replace `const numByes = activePlayers.length % 4` with `const { podSizes, byeCount: numByes } = computePodSizes(activePlayers.length, allowPodsOf3)`
    - Add warning for n=5 + allowPodsOf3: `'5 players cannot be split into pods of 3 and 4. Using 1 pod of 4 with 1 bye.'`
    - Replace `const numPods = podPlayers.length / 4` with using `podSizes` from computePodSizes
    - Pass `podSizes` (not `numPods`) to `greedyAssign(pool, podSizes, opponentHistory)`
    - Random chunk assignment: replace `pool.slice(i * 4, (i + 1) * 4)` with offset-based slicing using podSizes
    - Seat assignment: replace `shuffleArray([1, 2, 3, 4])` with `shuffleArray(Array.from({ length: finalPods[i].length }, (_, j) => j + 1))`
    - Replace `numPods` usage in bye pod_number with `podSizes.length`

    ## Step 4: Update existing tests

    - Update greedyAssign tests to pass `podSizes` array instead of `numPods` (e.g., `[4, 4]` instead of `2`)
    - Add new `computePodSizes` describe block with exhaustive tests for all player counts 3-20
    - Add `generatePods with allowPodsOf3` describe block
    - Add parameterized integration tests using `it.each` for player counts 4-20 in both toggle states
