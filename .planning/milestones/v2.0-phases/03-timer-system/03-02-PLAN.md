**1. Create `src/hooks/useCountdown.ts`:**

A hook that takes a RoundTimer and returns live countdown state, updating every second.

```typescript
import { useState, useEffect, useRef } from 'react'
import type { RoundTimer } from '@/types/database'

export interface CountdownState {
  /** Total remaining seconds (negative = overtime) */
  remainingSeconds: number
  /** Formatted display string: "45:30" or "+2:30" for overtime */
  display: string
  /** true when timer has passed zero */
  isOvertime: boolean
  /** true when timer is paused */
  isPaused: boolean
  /** true when timer is cancelled */
  isCancelled: boolean
  /** Urgency level for styling */
  urgency: 'normal' | 'warning' | 'danger' | 'expired'
}

export function useCountdown(timer: RoundTimer | null): CountdownState | null
```

Logic:
- If timer is null or cancelled, return null
- If timer.status === 'paused': remaining = timer.remaining_seconds (static, no interval)
- If timer.status === 'running': remaining = Math.floor((new Date(timer.expires_at).getTime() - Date.now()) / 1000)
- Use `setInterval(1000)` for live updates when running. Clear interval on unmount or when paused.
- Use `useRef` for the interval ID. Use `useState` for remainingSeconds.
- **Urgency calculation:**
  - remaining > 600 (10 min): 'normal'
  - remaining > 300 (5 min): 'warning'
  - remaining > 0: 'danger'
  - remaining <= 0: 'expired'
- **Display formatting:**
  - If remaining >= 0: `${Math.floor(remaining / 60)}:${(remaining % 60).toString().padStart(2, '0')}`
  - If remaining < 0: `+${Math.floor(Math.abs(remaining) / 60)}:${(Math.abs(remaining) % 60).toString().padStart(2, '0')}`
- Pad minutes to at least 1 digit (no leading zero for minutes): "5:30", "90:00", "+2:30"
